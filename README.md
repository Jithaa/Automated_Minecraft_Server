## Developed for Personal and Educational Purpose

    which is then Port Forwarded allowing people(/Friends) to connect

    Game-Any OS

    -time
    -threading
    -Urlib
    -os
 General:
    -Java
    -Ngrok
    -Server Startable Batch file name "ser"
